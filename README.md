# u2json
![Go](https://github.com/saka1/u2json/workflows/Go/badge.svg)

A Utility to convert URL to JSON containing each parts.

```shell
$ u2json 'https://example.com:80/foo/bar'
{"host":"example.com","path":"/foo/bar","port":80,"scheme":"https"}
```
